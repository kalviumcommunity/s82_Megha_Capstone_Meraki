# Meraki Platform

Meraki is a modern platform designed to bridge the gap between passionate volunteers and community-driven organizations (NGOs). Our mission is to amplify social impact through seamless collaboration, transparent tracking, and purpose-driven engagement.

---

## 🏗 Project Structure

This project follows a monorepo-style architecture with separate directories for frontend and backend.

### [Frontend](./frontend)
- **Tech Stack**: React, Vite, Tailwind CSS, Lucide Icons.
- **Key Features**: Multi-step onboarding, Impact Portfolio, Opportunity Explorer, Organization Dashboard.
- **Run**: `cd frontend && npm install && npm run dev`

### [Backend](./backend)
- **Tech Stack**: Node.js, Express, CORS, Dotenv.
- **Key Features**: Authentication placeholders, Opportunities API, User Profile management.
- **Run**: `cd backend && npm install && npm run start`

---

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1. **Clone the repository** (if not already done).
2. **Install Frontend dependencies**:
   ```bash
   cd frontend
   npm install
   ```
3. **Install Backend dependencies**:
   ```bash
   cd backend
   npm install
   ```

### Running Locally

**To run the Frontend:**
```bash
cd frontend
npm run dev
```

**To run the Backend:**
```bash
cd backend
npm run start
```
